# agent.md

## 프로젝트
이 프로젝트는 Python + Streamlit으로 만드는 카드 뒤집기 기억력 게임이다.
초보자 수업용 starter 프로젝트로 사용한다.

## 목표
초기 버전에서는 아래 기능만 구현한다.
- 4x4 보드
- 8쌍 카드 섞기
- 카드 클릭 시 뒤집기
- 한 번에 두 장 선택
- 같으면 유지, 다르면 다시 닫기
- 모든 카드를 맞히면 종료 메시지 표시

## 제외 기능
초기 버전에서는 아래 기능을 구현하지 않는다.
- 타이머
- 점수
- 최고 기록 저장
- 난이도 선택
- 힌트
- 2인용 모드

## 기술
- Python
- Streamlit

## 파일 구조
아래 구조를 기본으로 프로젝트를 생성한다.
- app.py
- game_logic.py
- state.py
- requirements.txt
- README.md
- agent.md

## 구현 원칙
- 구조는 단순하게 유지한다
- 초보자가 읽기 쉽게 작성한다
- 불필요한 파일이나 복잡한 패턴은 피한다
- 화면 로직과 게임 로직은 가능한 분리한다
- 함수는 짧고 역할이 분명해야 한다
- 변수명은 이해하기 쉽게 작성한다
- 필요한 곳에만 짧은 주석을 추가한다

## 상태 관리
최소한 아래 상태를 명확하게 관리한다.
- 카드 목록
- 현재 선택된 카드
- 이미 맞춘 카드
- 현재 뒤집힌 카드
- 게임 종료 여부

## README 규칙
README.md는 사용자용 문서다.
GitHub 저장소 첫 화면에 올려도 자연스럽도록 작성한다.

README에는 최소한 아래 내용을 포함한다.
- 프로젝트 소개
- 실행 방법
- 파일 구조
- 현재 구현된 기능
- 이후 확장 아이디어

중요:
- 코드 구조나 기능이 바뀌면 README.md도 함께 수정한다
- README 내용은 실제 코드와 항상 일치해야 한다
- README에는 구현되지 않은 기능을 구현된 것처럼 쓰지 않는다

## 작업 방식
항상 아래 순서로 작업한다.
1. agent.md를 읽고 요구사항을 이해한다
2. 먼저 짧은 계획을 제시한다
3. 프로젝트 파일과 초기 코드를 생성하거나 수정한다
4. README.md를 실제 코드 상태에 맞게 작성하거나 수정한다
5. 마지막에 생성/수정한 파일과 핵심 내용을 요약한다

- 두 장 비교가 되는가
- 같으면 유지되는가
