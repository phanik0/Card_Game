# 🃏 카드 뒤집기 기억력 게임 (Memory Match Game)

Python과 Streamlit을 활용하여 만든 간단한 카드 뒤집기 기억력 게임입니다. 
누구나 쉽게 이해하고 수정할 수 있도록 로직을 명확하게 분리한 초보자 수업용 Starter 프로젝트입니다.

## 🌟 프로젝트 소개
4x4 보드에서 8쌍의 그림 카드(이모지)의 짝을 모두 맞추는 클래식한 브라우저 기반의 기억력 게임입니다.
카드를 한 번에 두 장씩 뒤집어가며 같은 그림을 찾고, 모든 카드의 짝을 맞추면 성공적으로 게임이 종료됩니다.

## 🚀 실행 방법

### 1. 패키지 설치
이 프로젝트는 Python 환경과 Streamlit 라이브러리가 필요합니다. 폴더 내에서 아래 명령어로 라이브러리를 설치해 주세요.
```bash
pip install -r requirements.txt
```

### 2. 게임 실행
아래 명령어를 터미널에 입력하여 Streamlit 웹 애플리케이션을 실행합니다.
```bash
streamlit run app.py
```

### 3. 게임 플레이
- 브라우저가 열리면 화면의 ❓ 카드를 클릭하세요.
- 첫 카드를 뒤집고, 두 번째 카드를 뒤집었을 때 두 그림이 같으면 짝이 맞은 것입니다.
- 짝이 다르면 약 0.5초 뒤에 카드가 자동으로 다시 닫힙니다.
- 총 16장(8쌍)의 카드를 모두 매치시키면 승리 메시지를 확인할 수 있습니다! 전진!

## 📂 파일 구조
코드의 일관성을 유지하고, 시각적 부분과 논리 부분을 명확히 분리하였습니다.

```text
.
├── app.py              # 전체 게임 UI 구성 및 앱 실행의 진입점(Entry Point)
├── game_logic.py       # 게임 보드 데이터 생성, 카드 섞기, 매칭 비교 등 핵심 로직
├── state.py            # Streamlit의 특징인 session_state 초기화 및 초기화 설정
├── requirements.txt    # 실행에 필요한 파이썬 패키지(Streamlit)
├── README.md           # 현재 읽고 계신 프로젝트 소개용 문서
└── agent.md            # 본 프로젝트를 생성할 때 기준으로 삼은 규칙 및 요구사항 문서
```

## ✨ 현재 구현된 핵심 기능
- **4x4 카드 보드**: Streamlit의 Columns 기능을 이용해 직관적인 그리드 형태로 구성
- **게임 엔진**: 8개의 과일 세트를 랜덤하게(Random Shuffle) 섞어 보드에 배치
- **안전한 상태 관리**: 카드가 계속 뒤집히거나, 이미 맞춘 카드가 닫히지 않는 안전한 `session_state` 구조
- **지연 이펙트**: 두 번째 카드를 뽑았을 때 `time.sleep`을 활용해 사용자가 카드를 인지할 시간을 부여
- **모든 카드 매칭 완료 알림 및 초기화 기능** 포함
